## kb-public

Публичная часть базы знаний.

### Содержимое

- `knowledge/` — заметки/карточки/спеки (публичные).
- `docs/` — документация репозитория (как пользоваться, как публиковать).

### Публикация

Этот репозиторий задуман как **публичный mirror**: сюда попадает только то, что не содержит приватных данных.
