## Ключевые идеи кластера

- **Паттерн** — это инструкция для *движка*: буквальные символы, метасимволы, квантификаторы и утверждения (zero-width).
- **Несовпадение диалектов** — одна и та же строка может вести себя иначе в JavaScript, PCRE, Java, .NET.
- **Стоимость** — NFA с обратным поиском может экспоненциально перебирать варианты; лечится структурой паттерна и инструментами диалекта (атомарность, владение, таймаут).
