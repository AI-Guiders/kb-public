# Russian language pedagogy — fundamentals v1

**Мир:** `pedagogy.russian-language`  
**Версия:** v1.0 · 2026-06-06 · **status:** active

---

## Scope

Written mastery: fluent read, orthography/punctuation in context, dialogue. Out of scope v1: audio, orthoepy, school linguistics exam.

---

## North star

**SVR + orthographic mapping:** build **D** (word recognition on field) and **C** (sentence/passage comprehension) with held-out probes.

Evidence: [`kb-pedagogy-ru-scientific-evidence-v1.md`](kb-pedagogy-ru-scientific-evidence-v1.md).

---

## Four pillars (written v1)

| Pillar | Construct | Probe idea |
|--------|-----------|------------|
| L1 | Lexical fluent read | whole-word held-out |
| L2 | OOV / analytic bootstrap | fade scaffold |
| L3 | Orthography in context | spelling in sentence |
| L4 | Punctuation / text dialogue | compose + readback |
