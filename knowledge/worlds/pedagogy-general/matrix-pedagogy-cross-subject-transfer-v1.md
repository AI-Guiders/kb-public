**world:** `pedagogy.general`

## Purpose

Перенос **instructional principles** между школьными предметами без импорта школьной программы как истины.

- Переносить principles (Rosenshine, CLT), не «тему урока».

## Matrix A — Instruction science → all subjects

| Pattern | Core | Gate check | Boundary |
|---------|------|------------|----------|
| Small steps | one construct/moment | single-skill probe | no meaningless micro-drill |
| Worked example | model before problem | meta-card / contrast | expertise reversal |
| Split-attention | one primary channel | vis OR text | no duplicate paste |
| Held-out | generalization | new item same skill | not recap same paragraph |
