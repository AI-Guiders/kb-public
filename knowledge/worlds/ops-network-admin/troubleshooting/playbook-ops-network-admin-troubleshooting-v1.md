## SSH / SCP

| Симптом | Проверить |
|---------|-----------|
| Не подключается | `ssh -v`; права `~/.ssh` (700), `authorized_keys` (600) |
| SCP падает | Права на целевой каталог, квота диска |
| Agent forwarding / bastion | `ssh-add -l`; не копировать приватный ключ на промежуточный хост без нужды |

**Ложные гипотезы:** «сменить порт sshd на проде без change»; парольный вход на внешних серверах.

---

## nginx / PHP-FPM

| Симптом | Проверить |
|---------|-----------|
| 5xx / таймаут upstream | `error.log`, время ответа upstream |
| Очередь / медленные ответы | Пул PHP-FPM, `stub_status`, метрики FPM |

---

## Wireshark

| Симптом | Проверить |
|---------|-----------|
| «Ничего не видно» | Интерфейс, promiscuous, трафик на сегменте, фильтр capture |
| Задержки | Delta между пакетами, ретраны, RST |

## 1С (администрирование)

| Симптом | Проверить |
|---------|-----------|
| Документы не проводятся | Очередь непроведённых, логи проведения |
| Таймауты | Блокировки, нагрузка на СУБД |
| Регламент «висит» | Время регламентных заданий vs нагрузка |
