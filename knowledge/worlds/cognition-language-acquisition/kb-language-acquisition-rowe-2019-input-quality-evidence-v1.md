## Fundamentals

**Thesis:** High-quality caregiver input is **interactionally supportive**, **linguistically adapted**, and **conceptually challenging** — not merely more words.

| Dimension | Examples |
|-----------|----------|
| **Interactive** | Contingency, joint attention, recasts/reformulations |
| **Linguistic** | Lexical diversity, syntactic complexity, decontextualized talk |
| **Conceptual** | Cognitive challenge, explanations, book talk beyond labels |

**2026 role:** operationalizes Anderson 2021 «quality» for **parent/educator messaging** and SNL home-language policy — still **not** school probe oracle alone.
