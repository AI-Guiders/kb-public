# Medicine (Evidence-Based) Playbook v1

## Purpose

Операционный контракт для запросов о BCI и смежных нейрореабилитационных направлениях: когда загружать evidence-документ, как не переходить границу «совет/назначение».

## When To Load

- Запросы о **BCI** (brain-computer interface), нейроинтерфейсах, имплантах для коммуникации/движения.
- Запросы о **нейрореабилитации** по доказательной базе: инсульт (двигательная реабилитация руки), травма спинного мозга, locked-in синдром, БАС, восстановление речи при параличе, ДЦП/дети, нейрофидбек (EEG).
- Запросы «evidence-based по этому направлению медицины» в указанных областях.

→ Загружать `kb-bci-evidence-based-medicine-v1.md`.

## Epistemic Contract

- Уровни доказательности и ссылки — из документа; не приписывать клиническую силу без указания на тип исследования (RCT, мета-анализ, серия случаев).
- Клинические решения остаются за руководствами и локальными протоколами; документ — навигация, не замена суждению.

## Do-Not-Overreach

- Не формулировать назначения/противопоказания «на пациента».
- Не обобщать с одного обзора на «всегда/никогда» без отсылки к уровню доказательности из таблицы документа.

## Integration

- Index: `index-knowledge-router-v1.md` → Domain Entry Map, строка Medicine (Evidence).
- Связь с эпистемией: `epistemic-playbook.md` (проверка утверждений, недоверие по умолчанию) применима к любым заявлениям о «эффективности» вне ссылки на документ.
