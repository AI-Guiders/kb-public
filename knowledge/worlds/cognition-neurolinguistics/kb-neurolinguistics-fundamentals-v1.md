## Anti-patterns (neuromyth deny list)

| Myth | Reality |
|------|---------|
| «Language only in left hemisphere» | Distributed network; reading recruits visual + phonological + semantic systems |
| «Critical period ends at 7» | School L2 (Genesee) + Hartshorne boundary — **different constructs**; see boundary card |
| «Bilingualism confuses the brain» | Control network adapts; school bilingual evidence ≠ harm (see pedagogy SNL) |
| «fMRI proves method X» | Neuro **converges** with behavioral phonics (Dehaene); gates stay behavioral |
| «Reading is purely visual whole-word» | VWFA requires **print–speech mapping** training; visual exposure alone insufficient |
