## Fundamentals

**Thesis:** L1 and L2 largely share a **common neural network**; L2 processing **additionally recruits executive control** beyond L1, especially when L2 acquired later.

**ALE meta-analysis — key patterns:**

| Level | L1 vs L2 pattern |
|-------|------------------|
| **Lexico-semantic** | L1 engages **wider** cortico-subcortical system when L2 learned late |
| **L2 processing** | Exceeds L1 semantic network + **executive control** regions |
| **Grammar / phonology** | Few regions **selective** — mostly shared |
| **Language switching** | Overlap domain-general control + bilingual control networks |

**2020+ update role:** quantitatively consolidates Perani/Green narrative before Yeh 2026 developmental review.
