## Fundamentals

**Thesis:** behavioral reading intervention works; **brain changes are real but heterogeneous** — PRISMA review of **39** neuroimaging studies; coordinate-based meta on **8** fMRI pre/post studies (**null** significant cluster convergence).

**Qualitative synthesis (consistent themes):**

| Pattern | Description |
|---------|-------------|
| **Normalization** | RD/at-risk groups shift toward typical reading-network activation (VWFA, temporal, frontal) post-intervention |
| **Compensation** | Additional regions (cerebellar, subcortical) — alternate routes when core network immature |
| **Intervention content** | Studies used **phoneme mapping, GPC, grapheme–phoneme training** — not whole-word-only |
| **VWFA multiplex** | Left occipito-temporal changes discussed via Chen et al. 2019 multiplex model — orthography + phonology + attention |

**Honest null meta:** small n, heterogeneous paradigms → **no single ALE peak** — does **not** mean interventions don't change brains; means **methodology immature for one-size coordinate**.

**2022+ implication:** neuro **supports** continued behavioral stack (Castles, intervention RCTs); cannot yet pick winners by fMRI biomarker alone.
