## Fundamentals

**Thesis:** K-12 CS is organized as **five core concepts** enacted through **seven practices** — not a syntax-first language course.

**Five concepts:**

1. Computing Systems
2. Networks and the Internet
3. Data and Analysis
4. Algorithms and Programming
5. Impacts of Computing

**Seven practices (selection):** fostering inclusive computing culture; collaborating; recognizing computational problems; **developing and using abstractions**; creating computational artifacts; testing/debugging; communicating about computing.

**Design principle:** concepts and practices **co-develop** across grade bands (K-2, 3-5, 6-8, 9-12) with increasing abstraction — ethics/impacts integrated, not bolt-on.
