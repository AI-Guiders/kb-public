# Informatics pedagogy — fundamentals v1

**Мир:** `pedagogy.informatics`  
**Версия:** v1.0 · 2026-06-06 · **status:** draft

---

## Scope

School informatics / CS: practices, concepts, PRIMM programming progression, digital literacy. Out of scope v1: enterprise devops, production security ops as gate.

---

## North star

**Computational thinking + PRIMM:** learners **predict, run, investigate, modify, make** — practices over syntax drill alone.

Evidence: [`kb-pedagogy-informatics-scientific-evidence-v1.md`](kb-pedagogy-informatics-scientific-evidence-v1.md).

---

## Four pillars (v1)

| Pillar | Construct | Probe idea |
|--------|-----------|------------|
| I1 | Practices (algorithm, abstraction, etc.) | explain unfamiliar program |
| I2 | Concepts (data, control, systems) | transfer to new language surface |
| I3 | PRIMM progression | predict before run; investigate trace |
| I4 | Create / debugging discipline | modify to spec; test edge |

## Anti-patterns

- Line-by-line syntax quiz as mastery
- Skip predict/investigate; jump to open coding
- Framework buzzwords without practice assessment
