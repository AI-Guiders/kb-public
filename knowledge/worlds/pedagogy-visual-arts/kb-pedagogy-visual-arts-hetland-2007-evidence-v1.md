## Fundamentals

**Thesis:** rigorous visual arts instruction develops **Studio Habits of Mind** — thinking dispositions visible in **making, observing, and critique**, not only technical skill.

**Eight Studio Habits (canonical list):**

| Habit | Observable behavior |
|-------|---------------------|
| **Develop Craft** | Technique, tools, studio practice |
| **Engage & Persist** | Sustained focus through challenges |
| **Envision** | Mental plan before/during work |
| **Express** | Convey meaning/intent |
| **Observe** | Attend to visual detail |
| **Reflect** | Evaluate and explain choices |
| **Stretch & Explore** | Play, risk, «what if» |
| **Understand Art Worlds** | Domain, history, critique norms |

**Studio structures (instructional, not optional):** teacher demonstration → sustained work time → **critique** (describe/analyze/evaluate) → reflection journal.
