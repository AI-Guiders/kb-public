# Second-native language pedagogy — fundamentals v1

**Мир:** `pedagogy.second-native-language`  
**Версия:** v1.0 · 2026-06-06 · **status:** draft

---

## Scope

**Второй (и последующие) родной язык** в школьной программе von Neumann: EN v1, та же рамка для любого языка.

**In scope:** bilingual/immersion/dual-language time; L1→L2 interdependence; oral acquisition environment; **written literacy wave** для alphabetic L2 (EN).

**Out of scope v1:** adult FL classroom model; exam-prep translation-only; literary stylistics at native depth; phonics для logographic L2 без отдельной orthography card.

**Anti-pattern (explicit):** «иностранный язык 2 урока в неделю, grammar translation» — **не** этот мир.

Evidence: [`kb-pedagogy-snl-scientific-evidence-v1.md`](kb-pedagogy-snl-scientific-evidence-v1.md).

---

## North star

**Bilingual native-like competence:** сильный **L1 (RU)** + **long-term dual-language/immersion** exposure → L2 (EN) как второй родной, не как «школьный иностранный».

Stack: Cummins interdependence → Thomas & Collier dual-language dosage → **Genesee immersion** (content + counterbalanced form-focus) → L1 acquisition bridge (input/recasts) → alphabetic written track (Castles bridge for EN).

---

## Program arc (von Neumann EN v1)

| Wave | Track | Content | Gate |
|------|-------|---------|------|
| **0** | L1 anchor | RU literacy on grade (see `pedagogy.russian-language`) | Cummins: L1 CALP before L2 literacy acceleration |
| **1** | Oral L2 environment | Dual-language / rich input hours; communication-first | Comprehension + production in novel context |
| **2** | EN written (alphabetic) | Castles §1 code → Ehri OM → §4 comprehension (EN bridge card) | D+C; print path |
| **3** | Disciplinary EN | Shanahan-style disciplinary literacy in English where needed | Text-evidence argument |

**FoF / form attention:** как в L1 — **эпизоды** в meaningful discourse (Long), не grammar-first course (см. Krashen card — immersion input, не FL lab).

---

## Four pillars (v1)

| Pillar | Construct | Probe idea |
|--------|-----------|------------|
| S1 | L1–L2 interdependence | L1 literacy + L2 oral before forced L2 literacy rush |
| S2 | Dual-language dosage | documented % L2 time (target: long-term ≥50% for quality models) |
| S3 | Acquisition-rich input | hours of meaningful L2, i+1 band |
| S4 | Written L2 (if alphabetic) | same evidence chain as RU print, language-specific orthography |

---

## Anti-patterns

- Grammar rule test as sole mastery
- L2 literacy before L1 CALP threshold (Cummins)
- Short transitional bilingual only (<6 yr) as «done»
- Deny L1 acquisition bridge while claiming «native-like»
- Classroom FL 2×45 min without environment hours
