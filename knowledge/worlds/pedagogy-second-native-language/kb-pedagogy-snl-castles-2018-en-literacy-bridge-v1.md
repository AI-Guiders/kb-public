## Fundamentals

**Thesis:** for **English as second-native written language**, use the **same program arc** as RU native literacy, with EN grapheme-phoneme inventory:

| Stage | EN application (second-native) |
|-------|-------------------------------|
| **Wave 0 — Code** | Systematic phonics / decoding (Castles §1); not sight-word-only |
| **Wave 1 — OM** | Orthographic mapping fluency (Ehri 2014) |
| **Wave 2 — Comprehension** | Castles §4; SVR gates D+C |
| **Prerequisite** | Cummins L1 CALP anchor — RU reading on grade |

**Difference from RU card:** phoneme-grapheme set, morphology (EN prefixes/suffixes), vocabulary frequency — **language-specific authoring**, same evidence principles.

**Difference from FL classroom:** written EN taught as **literacy acquisition in L2 environment**, not grammar translation + reading passages.
