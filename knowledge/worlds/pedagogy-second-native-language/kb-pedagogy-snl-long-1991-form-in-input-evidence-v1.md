## Fundamentals

**Thesis:** **Focus on Form (FoF)** — brief attention to linguistic form **during** meaningful communication — mirrors how caregivers redirect child speech in L1.

**Contrast (second-native):**

| Mode | Second-native stance |
|------|---------------------|
| **Focus on Forms** (grammar syllabus) | **Deny** as program spine — FL classroom artifact |
| **Meaning-only immersion** | Insufficient — allow FoF episodes like L1 recasts |
| **FoF in discourse** | **Allow** — communication breakdown → reformulation |

**Bridge to cognition L1:** recast, expansion, clarification request — same **class** of moves, applied in L2 medium by teachers/peers.
