# Transfer matrix — second-native language ↔ L1 acquisition v1

**Мир:** `pedagogy.second-native-language`  
**См. также:** [`../cognition-language-acquisition/matrix-language-acquisition-transfer-v1.md`](../cognition-language-acquisition/matrix-language-acquisition-transfer-v1.md)

## Purpose

Зафиксировать **bridge**, а не deny: второй родной язык опирается на **те же acquisition-механизмы**, что child L1 (input, recasts, lexicon principles), но в **школьной bilingual/immersion** обёртке с сильным L1 (RU) фундаментом.

## Matrix

| Source | Target | Default | Boundary check |
|--------|--------|---------|----------------|
| `cognition.language-acquisition` | `pedagogy.second-native-language` | **allow-with-check** | Input quality, recast/clarification — **design**, не engineering oracle |
| `cognition.language-acquisition` CDS/recast cards | SNL oral environment design | **allow-with-check** | Не импортировать Saxton negative-evidence как автоматический gate |
| `pedagogy.russian-language` | `pedagogy.second-native-language` written track | **allow-with-check** | Тот же alphabetic stack (Castles §1→OM→§4) **если** орфография языка alphabetic; RU≠EN 1:1 grapheme |
| `cognition.neurolinguistics` | SNL oral + written rationale | **allow-with-check** | [`../cognition-neurolinguistics/matrix-neurolinguistics-pedagogy-bridge-v1.md`](../cognition-neurolinguistics/matrix-neurolinguistics-pedagogy-bridge-v1.md) |
| `pedagogy.second-native-language` | `cognition.language-acquisition` | **deny** | School bilingual ≠ infant L1 lab claims |
| Classroom FL pedagogy (grammar-first, translation) | SNL program | **deny** | Явный anti-pattern |

## Operational bridge (von Neumann)

| L1 acquisition signal | Second-native use |
|----------------------|-------------------|
| Rich modified input | L2 environment hours, not worksheet-only |
| Recast / expansion | In-conversation reformulation, not red-pen drill |
| Conventionality + contrast (Clark) | Vocabulary in **discourse**, not isolated lists |
| Cummins CALP | L1 literacy strength before/ac parallel to L2 literacy |

## Red flags

- «Krashen → no grammar ever» (нужны FoF-эпизоды как в L1 feedback)
- «Dual language без L1 time» (ломает interdependence)
- Импорт Castles в logographic L2 без orthography card
