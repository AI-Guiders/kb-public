## Fundamentals

**Thesis:** acquisition progresses through **comprehensible input (i+1)** in meaningful messages; conscious **learning** operates via Monitor under limited conditions.

**Second-native reframing (not FL):**

| FL misread | Second-native read |
|------------|-------------------|
| «No grammar ever» | Form episodes in discourse (Long); L1-style recasts |
| «Input class» 2 hrs/week | **Environment hours** as in dual-language/immersion |
| Textbook scope-sequence | Natural order awareness — don't block communication for rule order |
| Affective filter for exams | Low-anxiety **L2 medium** classrooms, interesting content |

**Five hypotheses:** acquisition–learning distinction; natural order; Monitor; Input; Affective filter — all map to **designing L2-rich school environment**, not worksheet substitution.
