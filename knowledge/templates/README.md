# `knowledge/templates/` — шаблоны карточек и артефактов

Здесь лежат **каркасы для копирования**, не живые экземпляры под один workspace.  
После копирования заполняй в согласованном месте (часто `work/projects/<scope>/…`).  
Контракт: **`../META/kb-taxonomy-v1.md`**.

| Файл | Назначение |
|------|------------|
| **`template-knowledge-card-v1.md`** | Карточка KB: Provenance, Metadata, Epistemic linkage, Core unit, Operationalization — **тот же шаблон**, что для evidence-карточек (`kb-*-evidence-v1`); отдельного `kb-evidence-template` нет. |
| **`template-domain-note-v1.md`** | Короткий каркас заметки под будущий `domains/<slug>/`. |

Копируй под новое имя в согласованное место (`knowledge/` для публикуемых kb-*, `work/projects/…` для оперативных черновиков — по политике).
