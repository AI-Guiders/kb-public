# `templates/cards/` — шаблоны карточек и domain-заметок

Каркасы для копирования в `knowledge/` или `work/projects/<scope>/`. Не редактировать «живые» экземпляры здесь.

| Файл | Назначение |
|------|------------|
| `template-knowledge-card-v1.md` | Универсальная KB-карточка (Provenance → Lifecycle) |
| `template-kb-evidence-v1.md` | **Evidence-профиль:** `kb-<topic>-evidence-v1.md` + Layer contract + Fundamentals |
| `template-domain-note-v1.md` | Короткая заметка под `domains/<slug>/` |

**Именование evidence:** `kb-<slug>-evidence-v1.md` — не отдельный «тип файла», а карточка по evidence-шаблону.

Пример заполнения: `worlds/evidence-humanities-shelf/kb-covey-7-habits-evidence-v1.md`.

Верхний индекс: [`../README.md`](../README.md). Матрицы переноса — [`../matrices/`](../matrices/README.md).
