## Контекст

Люди часто говорят образами и неявно. «В памяти есть зачатки интуиции» не обязана содержать слово «интуиция» — идея может быть в workstyle, execution-gate, refactor-after-check.

## Что делать

- Не ограничиваться точным grep. **route_context** по смыслу («проактивность», «улучшение по ходу») даже при образной формулировке.
- **Синонимы:** «зачатки интуиции» ≈ workstyle, минимальный безопасный шаг — искать по ним.
- **Пустой grep ≠ отсутствие.** Переформулировать, route_context, смежные секции (personal-workstyle, execution-gate, project-operational-memory-v1, README проекта).
- **Размытый запрос:** [`worlds/agent-orchestration/playbook-clarification-general-query-v1.md`](../../worlds/agent-orchestration/playbook-clarification-general-query-v1.md).

**Цель:** учитывать неявное высказывание, не только буквальные совпадения.
