# Playbook: строгое выполнение при отсылке к образцу (v1)

**Домен:** agent-operations  
**Секция hot:** `strict-execution-when-analogy-v1` *(stub при необходимости)*

## Принцип

«По аналогии с X», «как в X», «как finplan» → **строго**: структура и механизмы 1:1 по X, без импровизации.

## Что делать

- **Один проход проектирования** до кода: «идеальная раскладка по образцу X?»
- **Один источник правды:** если в образце INCLUDE — не дублировать текст ссылками «см. файл».
- **Имена и структура:** роль папки в образце (напр. `sources/`) → та же роль в целевом проекте.
