# Шаблон решения KB

Используй этот шаблон для решений, которые нужно переиспользовать.

| Поле | Что заполнить |
|---|---|
| Контекст | В какой ситуации возникло решение. Какие ограничения действуют. |
| Решение | Что именно решили делать. Почему это лучше альтернатив в текущих условиях. |
| Границы применимости | Где решение работает. Где его применять нельзя. |
| Проверка | Как проверить корректность применения. Какие сигналы говорят, что нужна корректировка. |
| Последствия | Что это решение упрощает. Какие риски/компромиссы добавляет. |
