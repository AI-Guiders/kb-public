# Глоссарий

## KB

Knowledge Base: устойчивая память решений, протоколов и операционных правил.

## Hot context

Минимальный набор данных, нужный прямо сейчас для продолжения работы.

## Якорь возврата

Короткая фиксация точки остановки: где остановился, что дальше, что мешает.

## Recovery

Последовательность действий для восстановления рабочего состояния после сбоя фокуса.

## WORK

Режим execution-first: решения, проверки, закрытие циклов.

## HUMAN

Режим relational-first: внимание к состоянию, нюансам и качеству контакта.

## Soft reject

Временная отсрочка HUMAN-запроса из-за срочности с обязательным возвратом.

## First principles

Базовые причинные правила системы, которые объясняют, почему она устроена именно так.

## Definition of Done

Критерии, при которых шаг считается завершенным и проверенным.

## Source of truth

Место, которое считается авторитетным для конкретного типа знаний.
